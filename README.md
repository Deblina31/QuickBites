# Quick Bites

Quick Bites is a static online food delivery and restaurant website built using HTML and CSS only.  
The project focuses on presenting a clean and user-friendly interface for browsing food items and restaurant offerings.

## Live Demo
https://quick-bites-lake.vercel.app/

## Overview
This project simulates a basic food ordering platform where users can explore menu items. It is designed as a frontend-only application without backend integration.

## Features
- Clean and structured user interface
- Restaurant-style landing page
- Food items and menu display
- Simple navigation across sections
- Lightweight and fast-loading design

## Tech Stack
- HTML5
- CSS3
  
## Limitations
- Not responsive across all devices
- No backend or database integration
- No real ordering or payment functionality
 
## Future Improvements
- Add responsiveness for mobile and tablet devices
- Integrate backend for order management
- Implement user authentication
- Add cart and checkout functionality

## Deployment
The project is deployed using Vercel.
